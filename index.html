Chunyen Huang

Web designer and UX designer

HTML

Basic programming language, almost all of the websites and its application are built using HTML to construct layouts and structures.

CSS

Cascading Style Sheets, major purpose of CSS is to change or modify style of different elements defined in certain HTML file.

JS

Javascript can increase the interaction between websites and users by logically setting functions, loops, array, object, and so on. JS is also the technology which link front-end and back-end development.

SASS

SASS is a kind of CSS pre-processor, it can edit and organize CSS code in a logical and easy way without redundant repeated code. SASS can also separate into many different file to reduce complexity of long CSS code.

Materialize

Built-in CSS and JS code can be easily used by putting corresponding classes to HTML elements. Specially useful when building Android-type application.

Illustrator

Useful tools for create simple icons, shapes. Moreover, it can export different types of file and then be used in many ways.

Photoshop

Powerful image, and photo editing tool. Photoshop not only can fine tune pictures, but also can create some amazing effects by combining different pictures or effects.

XD

XD is a straightforward prototype tool and easy to use. After finishing prototype, we can share the creations to others just by sending simple url link.

SVG

wide-used type of image file, SVG is vector type and wouldn't change resolution no matter what size of images be used. Also, the file size is smaller compare to other rector-type files.

Github

Powerful document version control tool. It can record every difference between different version and every modification. Once the file is recorded, users can even retrieve any version they want.
